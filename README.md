# Mini-SIEM

![Python](https://img.shields.io/badge/python-3.12-blue)

**Mini-SIEM** est un projet Python qui simule un petit système de gestion et de détection d’événements de sécurité (SIEM).  
Il permet de :  

- **Générer des logs** simulant différents événements systèmes et réseau  
- **Filtrer les faux positifs** pour ne conserver que les alertes pertinentes  
- **Visualiser les alertes** de manière simple et interactive  

---

## 🛠️ Prérequis

- **Python 3.12** ou supérieur  
- Modules Python : `pandas`, `numpy`, `plotly`, `matplotlib` (optionnel)  

Installer les dépendances :  

```bash
pip install pandas numpy plotly matplotlib
