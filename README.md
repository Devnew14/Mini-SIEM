# Mini-SIEM

![Python](https://img.shields.io/badge/python-3.12-blue)

Mini-SIEM est un projet Python qui simule un petit système de détection d’alertes (SIEM).  
Il permet de **générer des logs**, **filtrer les faux positifs**, et **visualiser les alertes** de façon simple et interactive.

---

## 🛠️ Prérequis

- Python 3.12 ou supérieur  
- Modules Python : `pandas`, `numpy`, `plotly`, `matplotlib` (optionnel)  

Installer les dépendances :  

```bash
pip install pandas numpy plotly matplotlib
